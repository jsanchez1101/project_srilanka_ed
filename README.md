## Student Final Project

Under the supervision and guidance of Dr Galathara Kahanda:

Designed and implemented the design and implementation of a full stack donation and e-learning platform focused on supporting education in needy areas of Sri Lanka. 
Designed + deployed a normalized MySQL database on AWS RDS to track donors, payments, webhook events, and transaction history.
Built using Node.JS and Express to handle Stripe checkout, webhook verification, and idempotent payment processing. Backend hosted on render.
Integrated frontend donation flows with backend APIs & implemented transparency features through structured data storage and reporting.
